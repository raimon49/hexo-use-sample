[How to create](http://raimon49.github.io/2015/04/25/create-blog-with-hexo.html)

Setup
-----

```sh
$ git clone git@github.com:raimon49/hexo-use-sample.git
$ cd hexo-use-sample
$ npm install
```

Write
-----

```sh
$ npm run create "write-article-title"
```

Publish
-------

```sh
$ npm run deploy
```
